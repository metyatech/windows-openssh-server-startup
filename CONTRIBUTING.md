# Contributing

Thanks for taking the time to contribute.

## Development setup
- Install PowerShell 5.1+ or PowerShell 7+.
- Install Pester and PSScriptAnalyzer if you plan to run tests and lint.

## Commands
- Lint: `./scripts/lint.ps1`
- Tests: `./scripts/test.ps1`
- Build: `./scripts/build.ps1`

## Pull requests
- Keep changes focused and include tests for behavior changes.
- Run lint/test/build locally before submitting.
